# 📐 Dimensions Viewer

Расширение для SketchUp, которое в реальном времени отображает габариты, площадь, объём и периметр выделенных объектов, а также умеет измерять и задавать зазоры между объектами с помощью лучей.

## Функционал:

**Измерения**
- Габариты ограничивающего параллелепипеда по осям X / Y / Z
- Локальные габариты для повёрнутых групп и компонентов (с учётом их собственных осей)
- Площадь поверхности (м²)
- Объём (м³)
- Периметр рёбер
- Копирование всех значений в один клик
- Автоматическое обновление при изменении выделения, а также при Undo / Redo

**Лучи**
- Построение лучей от граней объекта по шести направлениям (±X, ±Y, ±Z)
- Измерение расстояния до ближайшей преграды по каждому направлению
- Учёт групп и компонентов (инстансов) как преград — луч не проходит сквозь них
- Игнорирование собственной геометрии объекта (луч не «цепляется» сам за себя)
- Порог минимального зазора: если объект стоит вплотную, луч не строится
- Ограничение максимальной длины луча

**Режим предпросмотра (Preview Tool)**

- Интерактивная отрисовка лучей прямо во вьюпорте
- Подсветка луча при наведении курсора
- Клик по лучу — задать точное расстояние (зазор) до преграды
- Подписи с расстоянием в миллиметрах прямо на лучах
- Цветовая кодировка лучей по осям (X — красный, Y — зелёный, Z — синий)
- Выбор/добавление объектов кликом прямо в режиме предпросмотра (с модификатором Ctrl/Shift)

**Назначение «стен» (фильтр преград)**

- **«Скрыть всё, кроме выделенных объектов»** — через контекстное меню
- **«Показать всё»** — восстановление видимости всех объектов
Пункты доступны по правому клику мыши


## Установка
- Скачайте dimensions_viewer.rbz из раздела Releases.
- В SketchUp: Extensions → Extension Manager → Install Extension.
- Выберите скачанный файл .rbz.

## Использование

- Откройте Extensions → Dimensions Viewer.
- Выделите любой объект — окно обновится автоматически.
- Для измерения зазоров включите режим предпросмотра лучей.
- Кликните по лучу, чтобы задать точное расстояние до преграды.
- Для скрытия лишних объектов используйте контекстное меню (правый клик):
Скрыть всё, кроме выделенных объектов
Показать всё

## Требования

- SketchUp 2017 или новее
- Модель в миллиметрах (для корректных локальных габаритов и зазоров)
